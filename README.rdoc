== README

Steps:

1. clone Repo on any system 

<tt>rails should be installed</tt>

2. Go to app directory in terminal/cmd  

<tt>cd ajax-in-rails</tt>

3. Install required ruby gems 

<tt>bundle install</tt>

4. Migrate the database 

<tt>rake db:migrate</tt>

5. Create Demo posts for Post table

<tt>rake db:seed</tt>

6. Run rails server 

<tt>rails s</tt>

7. Go to browser to see working ajax with rails 

<tt>http://127.0.0.1:3000/</tt>

Have Fun :)
